# Phishing-Detection-System
Phishing is the threat that is so commonly spread and has the potential to cause huge damage. The criminals use fake emails to pretend to be the legit source and when they receive this sensitive data such as passwords, banking and identity details they are more than happy. The vast majority of users cannot recognize either fake sender address or infected content. This project is a witness to an endeavor to construct a smart detection system that will not only analyze the content of the email but also check the email headers (From, Reply-To, etc.) for traces of spoofing or impersonation to be able to make the system better equipped with merciless real-world cybersecurity tasks.
## Project Objective(s)
•	Detect the phishing emails by the help of a machine learning model that is used to train on both the content and the email headers.
•	Indicate that the headers have been spoofed by patterns such as the mismatch of "From" and "Reply-To" addresses.
•	Design and construct a web-based tool which serves for the purpose of testing and analysing the content of the email and its headers.
•	Accompany the feedback of phishing detection with tips to avoid unsafe practices when using email.

## Tools and Technologies
•	Programming Language: Python
•	Libraries: scikit-learn, pandas, TfidfVectorizer
•	Interface: Streamlit (for the web app)
